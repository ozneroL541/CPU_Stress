This program simulates different algorithms in order show how many steps they take to calculate the cartesian product between two tables. The program is parametrical respect the tables, R and S and the size of the buffer. The size of the buffer must be bigger then 3 and it\textquotesingle{}s consider only to perform multi-\/block algorithms, for the others it\textquotesingle{}s considered 3 block.\hypertarget{md_README_autotoc_md3}{}\doxysubsection{\texorpdfstring{Download}{Download}}\label{md_README_autotoc_md3}

\end{DoxyVerb}
