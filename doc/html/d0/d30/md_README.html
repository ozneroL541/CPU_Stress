Configuration

Change table length modifing the macros in include/tables.h. Change avaiable blocks in the buffer modifing the macro in include/buffer_blocks.

Compilation

‍It is suggested to use ICX Intel Compiler to compile this program, if it is not avaiable, GCC will be used.

To compile run: 

make

Execution

To execute the program run bin/blocks
